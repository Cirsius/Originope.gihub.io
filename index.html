Introduction

Valheim includes a developer console that allows you to enter cheat commands that can make you become invulnerable to damage, allow you to fly, remove building restrictions and spawn various items, among other things.
Accessing the developer console requires the -console launch option to be set in Steam before starting the game.

  • Right-click Valheim in your Steam library and select Properties
  • In the General tab, type -console into the Launch Options text box and close the Properties window

The developer console will now be enabled the next time you start Valheim.

  • Once in-game, the console can be opened by pressing the F5 key.
  • To enable cheat commands, type devcommands into the console and press Enter.
